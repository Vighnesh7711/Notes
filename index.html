Usage Note

Important for GitHub Pages:

This application uses a pre-defined list of PDF files that were available at creation. For this to work, ensure all listed files are located in the **same directory** as this index.html file in your GitHub repository.

If you add new PDF files to your repository, you will need to manually edit the list in the script section below.
